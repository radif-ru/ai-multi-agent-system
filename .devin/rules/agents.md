# Правила проекта

Источник истины — `AGENTS.md` в корне репозитория. Следуй ему всегда.

Скиллы-дисциплины — в `.agents/skills/<name>/SKILL.md`. Если тема задачи совпадает со скиллом, прочитай его **до** правок; для широкой задачи загружай `architecture-discipline` первым.

Проектные правила разработки — `_docs/instructions.md`; процесс задач — `_board/process.md`; архитектура — `_docs/architecture.md`.
